How to create react app

Post by Syed Tabish Ali, Sep 26, 2017.
react create-react-app

  • First install node JS in your workstation.
  • Nodejs
  • Make a folder in your workstation then open that folder and open the command prompt.
  • Run create-react-app app-name
  • It will take some time.
  • After the command successfully done then command prompt look like.
  • Then change your current directory into your app-name using cd app-name command.
  • Run the npm start command in your command prompt.
  • After few second react app has compiled and command prompt look it.
  • Your first app look like.
